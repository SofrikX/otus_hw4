# Scope текущего ДЗ — Frontend-приложение с AI-агентом

## Основа

Текущее ДЗ выполняется на основе проекта PetConnect из ДЗ 3 и техник Prompt Engineering из ДЗ 2.

## AI-агент

Используется OpenAI Codex.

Основной файл правил для агента: `AGENTS.md`.

## Стек

- Flutter
- Dart
- Riverpod
- go_router
- Material 3
- feature-first + Clean Architecture principles
- flutter_test
- mocktail

## Что входит в MVP

1. Лента публикаций питомцев.
2. Профиль питомца.
3. Прогулки и присоединение к прогулке.
4. Базовый экран чатов.

## Что не входит в MVP

- Реальная авторизация.
- Подключение Firebase.
- Firestore-запросы.
- Загрузка фото в Firebase Storage.
- Push-уведомления.
- Реальный backend.

Эти возможности остаются в целевой архитектуре, но не нужны для запуска frontend MVP.

## Обязательные артефакты сдачи

- GitHub-репозиторий.
- Код Flutter-приложения.
- `README.md`.
- `development_report.md`.
- `prompts.md`.
- `AGENTS.md`.
- Минимум 3 автоматических теста.
- Инструкции по локальному запуску.

## Критерии готовности

- Приложение запускается локально.
- Реализованы минимум 3 основные функции.
- Интерфейс корректно отображается на desktop и mobile.
- Есть автоматические тесты.
- Тесты проходят.
- Использование OpenAI Codex задокументировано.
- В документации нет путаницы между Cursor и Codex.
