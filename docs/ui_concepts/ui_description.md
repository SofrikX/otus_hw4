# UI-концепция PetConnect

Для текущего frontend MVP выбрана концепция №2: **яркая социальная сеть**.

## Причина выбора

Эта концепция лучше всего соответствует идее PetConnect как социальной сети для владельцев питомцев: она поддерживает эмоциональную ленту, карточки питомцев, визуальные акценты, быстрые действия и дружелюбный tone of voice.

## Визуальные ориентиры

- Material 3.
- Яркие, но не агрессивные акценты.
- Карточки с округлениями.
- Лента как основной экран.
- Быстрые действия: лайк, комментарий, присоединение к прогулке.
- Адаптивная центральная колонка на desktop.
- Удобная нижняя навигация на mobile.
